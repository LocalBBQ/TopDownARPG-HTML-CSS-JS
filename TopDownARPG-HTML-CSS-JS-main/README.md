A browser-based action RPG with top-down combat and dungeon-crawling. Vanilla JavaScript and HTML5 Canvas.

Download: On GitHub click Code → Download ZIP, then unzip the folder.

To run: Open index.html in your browser (Chrome, Edge, or Firefox). If something doesn’t work, run a simple server from the project folder instead — e.g. `python -m http.server 8000` then go to http://localhost:8000

Controls: WASD move, Shift sprint, Space dodge. Left click attack, right click block. R shoot, E portal.

No install or build. Just open and play.
